Title: Scheduling software. 
Purpose: A software program for C195 at WGU that utilizes Java and MySQL for an appointment management system.
Author: James Kuhr
Contact: jkuhr@wgu.edu
Version: 1.0 5/28/2023
IDE: IntelliJ Community 2021.1.3
JDK version: Java SE 17.0.1
JavaFX version: JavaFX-SDK-11.0.2
MySQL Connector: mysql-connector-java-8.0.25
Directions:
Launch program, enter username and password and submit to enter next screen. Acknowledge appointments occuring in the next 15 minutes.

Main usage:
All customers are displayed in the top table.  New customers can be added.  If a customer is selected in the table they can be modified or deleted. Selected customer appointments are listed in the lower table.
The "All Customers" button displays all customer appointments in the lower table.

Appointments are listed in the lower table.  Appointments can be added. If an appointent is selected it can be modified or deleted. Radio buttons will filter appointments based on if they occur, if they occur it the next month, or if they occur in the next week.

Reporting:
Reporting can be accessed via the "Reporting" button. It provides 3 reports: 1) Total number of appointments by type and month 2) total number of customers in a first level division 3) appointments sorted by contact.
